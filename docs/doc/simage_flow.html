simage_flow

项目介绍

用最简单最快捷的方式提供个人用户（公司用户）上传、分享、下载内容，通过平台连接人与内容、人与人、公司内部团队，加速内容的传播和提升人与人沟通的效率。内容包括图片、视频、工作文件等。此产品参考国外分享网站 wetransfer.com 。

针对人群：初期会针对设计师、摄影师等专业人士，以及中小团队作为团队内的内容存储和共享服务。上传分享相对专业的内容，而不是盗版影视内容。

产品目标：
第一阶段：实现上传、分享和下载这几个核心功能，支持下载预览、账户注册以及会员付费。

第二阶段：提升核心功能的体验，速度一定要快，建立优势，形成传播。

第三阶段：根据前期的用户反馈，研究产品的后续方向：偏大众的相册和分享/专业的内容存储和分享，以及协同编辑/针对企业客户的特殊服务/支持所有内容的预览，包括网页/支持上传内容的付费，开辟付费市场…等等

产品文档：

用户类型定义：

  1. 未注册用户（不可上传、可下载）
  2. 注册用户-免费会员（可上传、下载、分享，有功能限制）
  3. 注册用户-VIP会员（可上传、下载、分享，解锁功能限制）
  4. 注册用户-会员到期（可上传、下载、分享，有功能限制，到期前创建的分享均已失效）

会员权限细分：
免费会员：

  1. 单次上传最多1G，支持任何类型文件
  2. 分享链接7天后失效
  3. 加密传输
  4. 最多可分享给10个人
  5. 20G 总的存储空间

VIP会员：

  1. 单次上传最多到20G，支持任何类型文件
  2. 分享链接有效期可定制，可永久有效
  3. 加密传输
  4. 最多可分享给100个人
  5. 1000G 总存储空间
  6. 分享链接可设置访问密码
  7. 更多功能待开发

会员价格：

  1. 25元/月
  2. 200元/年

技术点：

  1. 文件和传输加密
  2. 上传和下载的速度到达极致
  3. 分享短链接
  4. 访问限制（有效期和密码）
  5. 内容预览功能
